Detailed Description

Internal functions and private definitions for the HDC3022 sensor driver.

This file contains internal implementations and macros intended for use within the HDC3022 driver only. It is not meant for direct use by application code.

Users should not include this file outside of the driver context.

Sends a 16-bit command to the HDC3022 sensor via I2C.

This function writes a 16-bit command to the HDC3022 sensor without any accompanying data. The nostop parameter controls whether a stop condition is generated after the transmission.

Sends a command and data to the HDC3022 sensor via I2C.

This function writes a 16-bit command followed by a 16-bit data value to the HDC3022 sensor. A CRC byte is calculated and appended to ensure data integrity. The nostop parameter controls whether a stop condition is generated after the transmission.

Parameters

[in]command16-bit command to send to the sensor.
[in]data16-bit data associated with the command.
[in]nostopIf true, no stop condition is sent on the I2C bus after writing.
Returns
true if the write operation was successful (5 bytes written), false otherwise.

Reads a single 16-bit data value from the HDC3022 sensor via I2C.

This function reads 2 bytes of data followed by a CRC byte from the sensor, verifies the CRC to ensure data integrity, and stores the received data. The nostop parameter controls whether a stop condition is generated after reading.

Parameters

Reads two 16-bit data values from the HDC3022 sensor via I2C.

This function reads 6 bytes from the sensor: two sets of 2-byte data values each followed by a CRC byte. It validates the CRC for both data values to ensure data integrity before storing them. The nostop parameter controls whether a stop condition is generated after reading.

Parameters
[in]hdc3022Pointer to the HDC3022 device structure.
[out]data1Pointer to a variable where the first 16-bit data will be stored.
[out]data2Pointer to a variable where the second 16-bit data will be stored.
[in]nostopIf true, no stop condition is sent on the I2C bus after reading.
Returns
true if both data values are successfully read and CRC validated, false otherwise.

Converts a temperature value to a raw 16-bit representation for the HDC3022 sensor.

This function takes a temperature value in a specified unit and converts it into the raw 16-bit format expected by the HDC3022 sensor. The conversion is based on the unit provided through hdc3022_TempUnit_t.

If the resulting raw value falls outside the valid 16-bit range (0–65535), it is saturated to the nearest boundary value.

Parameters
[in]tempUnitTemperature unit used for the input value. See hdc3022_TempUnit_t.
[in]tempTemperature value to convert.
Returns
The corresponding raw 16-bit temperature value for the HDC3022 sensor.

Converts a relative humidity value (RH) to a raw 16-bit format for the HDC3022 sensor.

This function converts a relative humidity value expressed in percent (RH) to the corresponding 16-bit raw format required by the HDC3022 sensor.
